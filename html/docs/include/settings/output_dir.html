The directory where Xdebug will write tracing, profiling, and garbage collection statistics to. This directory needs to be writable for the system user with which PHP is running.

This setting can be changed in php.ini, .htaccess (and equivalent files), and within a PHP file with ini_set(). In some cases (such as when [CFG:xdebug.auto_trace] is on, or while profiling), Xdebug creates the file before the script runs. In that case, changes made through ini_set() will not be taken into account.
